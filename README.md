# Biblioteca-mvc-Estefania-Carrillo-Jose-Miguel
Este proyecto consiste en el desarrollo de un sistema de gestión de biblioteca en Java siguiendo el patrón MVC (Modelo-Vista-Controlador). 
El sistema permite gestionar libros y usuarios, como realizar préstamos, devoluciones y reservas, controlando el estado y disponibilidad de los libros.

## Cómo ejecutar el programa
1. Abrir el proyecto en un IDE compatible con Java (Visual Studio Code, IntelliJ IDEA, Eclipse, etc).
2. Buscar la carpeta src/app/Main.java.
3. Ejecutar la clase Main.
